## 3. 핵심 인사이트 — 리뷰가 검증한 전략 진단

**🔴 가장 중요한 발견: 화해는 스스로의 유일한 해자(신뢰)를 훼손하고 있다.**
- 06-B1에서 화해의 **유일한 10배 차별점 = "판매와 분리된 객관적 성분 신뢰"** 로 진단했다.
- 그런데 충성 유저들이 직접 *"홈쇼핑이 되어버렸다", "구매 유도만 강화됐다"* 고 증언한다.
- 즉 화해는 적자 탈출을 위해 커머스를 밀었는데, 그 과정에서 **차별점의 원천(중립적 신뢰)을 자기 손으로 깎고 있다.** → 가장 위험한 자기모순.

**커머스 푸시의 역설 (정량+정성 통합, W1 §7)**
- 정량(07): 커머스 27억 영세 + 본업 적자 95억 → 커머스로 수익 내려는 압박.
- 정성(08): 유저는 "정가 수준·할인 축소"로 **화해 커머스를 외면**(가격은 쿠팡/올영이 낫다) + "홈쇼핑화"에 거부감.
- 통합 결론: **커머스 직접 강화는 ① 가격에서 못 이기고 ② 신뢰를 훼손하는 양패** → 05·06의 권고(개인화·광고/데이터 화폐화·글로벌)가 더 옳다는 방증.

## 4. 권고 (리뷰 근거로 보강)

1. **신뢰-커머스 분리 설계** — 커머스 푸시가 성분 신뢰 화면을 침범하지 않도록 UX 분리. "홈쇼핑화" 인식 차단이 최우선(코어층 이탈 방지).
2. **가격 경쟁 대신 개인화로 전환** — 할인으로 쿠팡·올영과 싸우지 말 것(못 이김). 리뷰가 요청한 **개인 맞춤 성분 추적·궁합 정렬·내 리뷰 검색** = 락인 기능으로 직결.
3. **기본 품질(버그) 복구** — 비교 로딩·튕김·임시저장은 코어층 불만 1순위. 신뢰 브랜드엔 안정성이 곧 신뢰.
4. **이탈층 리서치** — 평점 안 남기고 떠난 S2/S3 대상 별도 조사 필요(생존편향 보완).
