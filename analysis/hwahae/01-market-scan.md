## Executive Summary

화해는 38만 개 제품 성분 정보와 960만 건 리뷰, 1,300만 누적 다운로드를 보유한 한국 1세대 뷰티 정보 플랫폼이다. 2024년 매출 824억 원(전년비 +60%, 5년 CAGR 43.6%)으로 역대 최대를 기록했고 일부 분기 EBITDA 흑자를 달성했지만, 영업손실 약 28억 원·누적결손금 613억 원으로 **"성장하는 매출, 지속되는 적자"** 의 역설에 놓여 있다. 매출의 66%가 자회사(브랜드 비플레인)의 화장품 제조·판매에서 나와, 정보 플랫폼이 사실상 **자체 브랜드(PB) 회사로 무게중심이 이동**한 상태다. 핵심 위협은 ① 올리브영(MAU 1,038만, 4.7조 규모 슈퍼앱)의 정보 기능 내재화, ② Gen Z의 숏폼·감성 소비로의 이동, ③ "성분 투명화 완성 → 중개자 존재 이유 약화"라는 구조적 역설이다. 기회는 K-뷰티 수출 호황(대미 수출 1위 등극)을 활용한 **글로벌 웹 확장**과, 데이터 강점을 살린 **PB·안전성평가/e-라벨 규제 대응**에 있다. 전략 명제는 명확하다 — 단순 정보 중개자에서 벗어나 *데이터 기반 브랜드 빌더 + 글로벌 채널*로 재정의해야 한다.

## SWOT Analysis

| Strengths (내부 강점) | Weaknesses (내부 약점) |
|---|---|
| 38만 제품 성분 DB·960만 리뷰 = 모방 어려운 데이터 자산 | 영업적자 지속, 누적결손금 613억 — 수익 전환 효율 낮음 |
| '성분 신뢰' 브랜드 = 뷰티 고관여 유저, 높은 구매 전환율 | 매출 66%가 PB 자회사 의존 → 플랫폼 본업 수익성 취약 |
| 1,300만 누적 다운로드, 충성 코어 유저 기반 | 텍스트·성분 중심 UX → 숏폼/영상 트렌드에 뒤처짐 |
| PB(비플레인) 저자극 스킨케어 성공 사례 보유 | 커머스 규모(27억)가 올리브영 대비 미미, 유통 협상력 약함 |

| Opportunities (외부 기회) | Threats (외부 위협) |
|---|---|
| K-뷰티 수출 호황 — 대미 수출 22억 달러, 중국 추월 1위 | 올리브영 독점화(4.7조 슈퍼앱) + 정보 기능 내재화 |
| 글로벌 영문 웹 MAU, 론칭 2개월 만에 10만 돌파 | Gen Z 소비 패턴 변화: 성분 분석 → 인플루언서 15초 영상 |
| 2026 안전성평가·e-라벨 규제 → 성분 데이터 수요 증가 | 다이소 뷰티 등 초가성비 부상 → 정보 탐색 단계 단축 |
| 인디·중소 브랜드 확산 → 광고/입점 수요 | 쿠팡·네이버·무신사뷰티의 뷰티 커머스 공세 |

**SWOT 액션**
- **S+O (공격)**: 성분 DB를 영문화해 글로벌 웹 + K-뷰티 수출 파도에 결합, 데이터 기반 글로벌 큐레이션 채널화
- **W+T (방어)**: 텍스트 리뷰의 숏폼/영상 전환, PB 의존도 분산 + 플랫폼 커머스 수익화 구조 재설계

## Cross-Framework Synthesis

**수렴 신호 (여러 프레임워크가 공통 지목)**
- 화해의 핵심 자산인 '성분 정보 중개'는 **올리브영 내재화 + 정보 범용화 + 숏폼 전환**으로 가치가 동시 하락 (SWOT-T, Porter-대체재, PESTLE-S/T 일치).
- 실질 성장 동력은 이미 **PB(브랜드)와 글로벌**로 이동 (SWOT-O, Ansoff 제품개발·시장개발 일치).

**전략적 명제 (must-do)**
1. 포지셔닝 재정의: '성분 사전' → **데이터 기반 뷰티 브랜드 빌더 + 글로벌 큐레이션 채널**
2. 콘텐츠 전환: 텍스트 리뷰 → 숏폼·영상으로 Gen Z 재유입
3. 수익 구조 정상화: PB 단일 의존 완화 + 플랫폼 커머스/광고 전환율 무기화

**핵심 리스크**
- 올리브영 슈퍼앱 독점 가속 / 유저 전환비용 0(락인 부재) / PB 자회사 실적 변동 = 전사 실적 변동

**최적 기회 (리스크 조정)**
- 글로벌 영문 웹 × K-뷰티 수출(저비용·고성장) → 시장 개발이 가장 유리한 베팅
- 2026 안전성평가/e-라벨 규제를 데이터 강점으로 전환하는 B2B 솔루션

## Strategic Recommendations

1. **글로벌 우선(Global-first)** — 영문 웹의 초기 MAU 성장세(2개월 10만)와 K-뷰티 수출 1위 모멘텀을 결합, 한정 리소스를 국내 방어보다 글로벌 확장에 배분. *(Ansoff 시장개발 + SWOT S+O)*
2. **콘텐츠 모델 전환** — 텍스트 성분 리뷰를 숏폼/영상 + AI 추천으로 재구성해 Gen Z 이탈 방어. *(PESTLE-S/T + Porter 대체재)*
3. **데이터 머니타이제이션 다변화** — PB 의존(66%)을 낮추고, 성분 DB를 ① 인디 브랜드 광고·입점, ② 안전성평가/e-라벨 B2B 솔루션으로 양면 수익화. *(SWOT-W + 2026 규제 기회)*
4. **락인 설계** — 전환비용 0 문제 해결을 위해 개인 스킨 프로파일·구독·리워드 등 투자(Investment) 장치 도입. *(Porter 구매자 협상력)*
