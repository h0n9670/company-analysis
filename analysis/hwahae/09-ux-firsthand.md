## 2. 심층 진단

### U1 — 정체성 혼란이 첫 화면에서 발생 (가장 치명적)
- 화해의 **유일한 10배 차별점 = '성분 신뢰'**(06-B1)인데, **첫 진입 화면이 그 약속을 배신**하고 쇼핑몰을 보여준다.
- 08의 충성유저 증언("홈쇼핑이 되어버렸다")이 **신규 유저의 first-run에서도 동일하게 재현**됨 → 단발 불만이 아니라 **구조적 IA(정보구조) 문제**.
- 결과: "성분 보러 왔는데 왜 쇼핑몰?" → 기대-경험 불일치 → 즉시 이탈(W1 §즉각적 이탈) 위험.

### U2·U3 — 온보딩의 역설: 투자는 시켰는데 보상이 없다
- W1 §온보딩(02-market-sizing 사례): *긴 온보딩이 효과적인 건 "이 제품이 나에게 도움 될 것"이란 **기대를 형성**할 때만*이다.
- 화해는 설문(=유저의 시간·노력 **투자/Investment**)을 받아놓고, **그 투자를 개인화 추천·안내(=보상/Reward)로 되돌려주지 않는다.**
- Hook 모델(W1 §7) 관점: **Investment → (Trigger/Reward로 순환) 고리가 끊김** → 유저는 "헛수고했다"는 배신감 → 락인은커녕 이탈 가속.
- 즉, **허들(비용)은 다 치르게 하고 가치(보상)는 안 줌** = 최악의 온보딩 구조.

### U4 — 전문성 전제가 비전문 유저를 배제
- "건성/지성?", "어떤 피부 문제?"는 **이미 자기 피부를 아는 고관여자(S1)** 에겐 쉽지만, **대중·신규(S3, 캐즘 너머)** 에겐 답하기 어려운 질문.
- 화해의 가치는 "*몰라도* 좋은 화장품 고르게 돕는 것"이어야 하는데, 온보딩이 **이미 아는 사람**을 전제 → JTBD와 모순.
- → 진단 질문이 아니라 **점진적 학습형 유도**(쉬운 선택→결과를 보며 정교화)가 필요.

### U5 — 시각적 과부하: 정보 위에 색을 또 얹음
- 성분·리뷰·평점은 그 자체로 정보 밀도·색(등급 색상 등)이 높다. 여기에 **화려한 아이콘 색까지 더해 인지 부하(Brain Cycles, W1 능력 6요소) 증가**.
- 신뢰 브랜드일수록 **절제된 시각 위계**가 신뢰감을 준다 — 화려함은 '홈쇼핑/세일' 인상을 강화(U1 악화).

### U6 — UX 일관성 부재: 디자인 시스템의 부재 신호
- **로딩 인디케이터·인터랙션 패턴이 화면마다 제각각** → "난잡함". 이는 단순 미관 문제가 아니라 **공통 디자인 시스템/컴포넌트 규칙이 없다는 방증**.
- 일관성 결여는 ① 학습 비용↑(같은 동작인데 매번 다르게 보임, W1 Brain Cycles), ② **신뢰감 저하**(완성도 낮아 보임 — 신뢰가 핵심인 브랜드엔 치명적), ③ 개발·유지보수 부채 증가.
- 누적된 기능 추가(성분→리뷰→커머스→설문)가 **통합 설계 없이 덧붙여진 결과**로 추정 → U1(쇼핑몰화)과 같은 뿌리(증축식 성장).

### U7 — 홈 화면 모션 과다: 주의 분산과 시각 피로
- **움직이는 요소가 많아 눈이 피로** → 배너·자동 캐러셀·애니메이션이 주의를 분산시켜 **'어디를 봐야 할지' 위계가 무너짐**.
- 모션은 보통 **전환·구매를 유도하려는 커머스 의도**(눈길 끌기)에서 비롯 → U1·U5와 동일 맥락(커머스 푸시가 차분한 정보 탐색 경험을 침해).
- 정보 신뢰 앱의 핵심 과업("성분을 차분히 확인")과 **정면 충돌** — 움직임은 정독을 방해한다.

## 3. UX 처방 (개발자 관점 실행안)

| 이슈 | 처방 |
|---|---|
| U1 | **첫 화면을 '성분/내 피부 진단'으로** — 쇼핑몰이 아닌 핵심 가치(성분 신뢰)를 first-run에 배치. 커머스는 하위 탭으로 분리(08 권고 '신뢰-커머스 분리'와 동일). |
| U2 | **가치 먼저, 가입 나중** — 회원가입·설문 없이 **성분 검색 1회 체험(Aha moment)** 먼저 제공 → 가치 맛본 뒤 가입 유도(Fogg: Ability↑). |
| U3 | **설문 즉시 보상** — 설문 직후 "당신의 피부엔 이 성분 주의/추천" 같은 **개인화 결과를 즉각 노출**. 투자→보상 고리 복구(= 락인 기능, 05·06 권고). |
| U4 | **진단형 → 점진형 질문** — "잘 모르겠어요" 선택지 + 사진/증상 기반 쉬운 유도. 답을 강요하지 않고 사용하며 프로파일이 정교해지게. |
| U5 | **시각 위계 단순화** — 아이콘 채도↓, 색은 '성분 등급' 같은 의미 전달에만. 절제로 신뢰감·정보 가독성 확보. |
| U6 | **디자인 시스템 구축** — 로딩 인디케이터·버튼·카드·인터랙션을 공통 컴포넌트로 표준화. 일관성 = 완성도 = 신뢰감. |
| U7 | **홈 모션 절제** — 자동 캐러셀·애니메이션 최소화(또는 정지 옵션). 정적·차분한 정보 위주 홈으로 정독 환경 확보. |

## 4. 전략적 함의 (분석 종합과 연결)
- 이 1차 후기는 **00-종합보고서의 핵심 진단을 행동 수준에서 재확인**한다: *커머스 푸시가 신뢰(유일 차별점)를 첫 화면부터 훼손*.
- 특히 **U2·U3(온보딩 가치 미전달)** 은 05의 "S2 쇼루머·S3 Gen Z 이탈"이 *왜* 발생하는지의 **메커니즘**을 보여줌 — 가치를 느끼기 전에 허들에 막혀 떠난다.
- **U5·U6·U7(시각 과부하·일관성 부재·모션 과다)** 은 하나의 뿌리로 수렴: **통합 설계 없이 기능을 증축**(성분→리뷰→커머스)하며 *차분한 신뢰 경험*이 *화려한 커머스 자극*에 잠식됨. = U1과 동일 구조 문제의 시각적 발현.
- → 권고 우선순위 재확인: **① 신뢰-커머스 분리(첫 화면), ② 온보딩 Aha 모먼트 + 설문 즉시 개인화 보상**, 그리고 **③ 디자인 시스템 기반 시각·모션 절제**가 '신뢰 브랜드' 정체성 회복의 기반.
