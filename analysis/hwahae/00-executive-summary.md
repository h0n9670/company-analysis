## ③ 근본 원인 (5 Why → 시스템)
**"성분 정보 = 해자"라는 과거 성공 공식의 관성.** 정보가 범용화(올영 AI리뷰 흡수)된 뒤에도 비즈니스를 '정보 제공'으로 정의 → 가치 창출(정보)은 하나 가치 포착(거래·락인)을 설계 못 함.
**악순환(R1)**: 정보 공개 → 시장 평준화 → 경쟁사 흡수 → 쇼루밍(구매 이탈) → 커머스 정체 → (과거 PB로 메움 / 현재 적자 노출) → 재투자 위축 ↺

## ④ 증거 수렴 (분석 신뢰도 ↑)
모든 단계가 **동일 결론**으로 수렴:
- **경쟁(03)**: 올리브영(5.8조·MAU 1,053만) AI리뷰가 정보 해자 흡수 + 화해는 '정보 깊음/커머스 작음' 취약 구간
- **BM(04)**: 디커플링 피해('무료 쇼룸'), 커머스 27억 영세
- **차별화(06)**: 10배 우위 가능한 **유일 축 = 신뢰 × 개인화**
- **재무(07)**: PB 매각으로 본업 적자(95억) 노출 — 흑자화가 생존 과제
- **리뷰(08)**: 충성유저가 *"홈쇼핑이 되어버렸다"* — 커머스 푸시가 **신뢰(유일 차별점)를 자기훼손** + "정가라 경쟁력 없음"(쇼루머 이탈 확증)
- **직접 사용 UX(09)**: 첫 화면이 쇼핑몰(정체성 배신) + 가입·설문 허들은 높은데 **설문 결과를 개인화로 안 돌려줌**(투자→보상 단절) — 이탈 *메커니즘* 확인

## ⑤ 전략 방향 (우선순위)
1. **신뢰-커머스 분리** — 커머스 푸시가 성분 신뢰 경험을 침범하지 않게. "홈쇼핑화" 인식 차단(코어 이탈 방지).
2. **가격경쟁 → 개인화 락인** — 쿠팡·올영과 할인 싸움 회피. 개인 스킨 프로파일 × AI 성분 매칭 + 궁합 정렬·내 리뷰 검색(유저 요청).
3. **데이터·B2B·글로벌 화폐화** — 거래 못 가두면 광고·데이터로 수취. **화해 홀세일**(K뷰티 글로벌 등용문) = 회사가 이미 착수, 분석과 정합.
4. **기본 품질 복구** — 비교 로딩·튕김·임시저장 버그(코어 불만 1순위).
