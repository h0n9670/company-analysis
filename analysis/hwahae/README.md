## 한 장 요약 (Executive Synthesis)

**진단**: 화해는 매출 824억(+60%, 역대 최대)을 올리지만 영업적자·누적결손금 613억의 **"성장하는 매출, 지속되는 적자"** 상태. 매출 66%가 PB(비플레인) 제조에서 나와 사실상 화장품 제조사로 무게중심이 이동했다.

**근본 원인 (전 단계 공통 지목)**:
1. **정보 중개 가치 하락** — 올리브영(매출 5.8조, MAU 1,053만)이 4,700만 리뷰 + AI 요약으로 화해 핵심 가치를 흡수. (1·3단계)
2. **디커플링 피해** — 유저가 화해서 탐색·평가하고 구매는 올영/쿠팡에서. '무료 쇼룸' 구조, 커머스 27억 영세. (4단계)
3. **락인 부재** — 락인은 S1(성분 탐정)·S4(PB 구매자) 소수층뿐. 최대 풀 S2(쇼루머)를 화폐화 못 함. (5단계)

**시장 위치**: TAM 9~10조 / SAM 2조 / SOM 824억(SAM의 ~4%) — 시장은 크나 화해 몫은 1% 미만이고 잠식 중. (2단계)

**전략 방향 (우선순위)**:
1. **포지셔닝 재정의** — '성분 사전' → 데이터 기반 **개인화 매칭 + 글로벌 큐레이션 + 브랜드 빌더**
2. **S2 쇼루머 화폐화** — 구매 완결 전환 *또는* 광고·데이터 B2B 수취 (적자 해소의 핵심)
3. **콘텐츠 전환(S3) + 글로벌(S5)** — 숏폼·영상으로 Gen Z 유입, K뷰티 수출 파도에 영문 웹 결합
4. **PB 의존 분산 + 락인 장치** — 개인 스킨 프로파일·구독

**가장 유리한 베팅**: 국내 레드오션(올영 독점) 정면 경쟁 회피 + **글로벌 K뷰티 큐레이션(시장 개발)** — 저비용·고성장.
